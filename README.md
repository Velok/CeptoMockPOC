# CeptoMockPOC
POC for a C#/.NET mocking framework which uses SourceGenerators and Interceptors. The goal is to define mocks of objects which do not define a corresponding interface.

* How to setup debugging source code generators -> https://github.com/JoanComasFdz/dotnet-how-to-debug-source-generator-vs2022

* Interceptors Introduction -> https://andrewlock.net/exploring-the-dotnet-8-preview-changing-method-calls-with-interceptors/
